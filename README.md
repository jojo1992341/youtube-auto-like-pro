# YouTube Auto-Like Pro (v5)

[Français](#français) | [English](#english)

## Français

### Description
**YouTube Auto-Like Pro** est une extension Chrome conçue pour automatiser le processus de "Like" sur vos vidéos YouTube préférées. Refondue avec une architecture propre, modulaire et performante (V5), elle offre une expérience utilisateur fluide et fiable, y compris la gestion des commentaires assistés par IA.

### Fonctionnalités
- **Like Automatique** : Aime automatiquement les vidéos que vous regardez.
- **Architecture Propre** : Code structuré pour une maintenance facile et des performances optimales.
- **Interface Intuitive** : Une popup simple pour gérer vos préférences.
- **Sélecteurs personnalisables** : Possibilité de configurer les sélecteurs (boutons, chaîne, commentaires) pour s'adapter aux variations du DOM YouTube.

### Installation
1. Téléchargez ou clônez ce dépôt.
2. Ouvrez Chrome et accédez à `chrome://extensions/`.
3. Activez le **Mode développeur** (en haut à droite).
4. Cliquez sur **Charger l'extension décompressée** et sélectionnez le dossier du projet.

## English

### Description
**YouTube Auto-Like Pro** is a Chrome extension designed to automate the "Like" process on your favorite YouTube videos. Rebuilt with a clean, modular, and high-performance architecture (V5), it provides a smooth and reliable user experience, including AI-assisted comment handling.

### Features
- **Automatic Like**: Automatically likes the videos you watch.
- **Clean Architecture**: Structured code for easy maintenance and optimal performance.
- **Intuitive Interface**: A simple popup to manage your preferences.
- **Configurable selectors**: Customize selectors (buttons, channel, comments) to adapt to YouTube DOM variations.

### Installation
1. Download or clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select the project folder.
